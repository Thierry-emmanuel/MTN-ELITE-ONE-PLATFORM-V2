# 🇨🇲 MTN Elite One Digital Platform

A modern, professional and patriotic digital platform dedicated to the **Cameroonian MTN Elite One League**.

This project aims to become the **official digital home of Cameroonian football**, inspired by world-class platforms such as the Premier League, La Liga and Bundesliga.

---

## 🎯 Vision
Build a premium **web and mobile football platform** that showcases:

- fixtures
- results
- league table
- player statistics
- club pages
- Cameroonian Ballon d'Or
- team of the week / season
- football culture stories
- hall of fame
- young talent watch
- road to the lions

---

## 🚀 Tech Stack

### Frontend
- React + Vite + TypeScript
- Tailwind CSS
- React Router DOM
- Axios
- Zustand
- Framer Motion

### Backend (planned)
- NestJS
- Node.js
- Socket.io

### Database (planned)
- PostgreSQL
- MongoDB

### CI/CD
- GitHub Actions
- Vercel (frontend deployment)
- Render / Railway (backend deployment)
